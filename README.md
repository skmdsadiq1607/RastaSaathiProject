# RastaSaathi | AI-Powered Emergency Response 🚑✨

Built for the **IIT Madras Road Safety Hackathon 2026**, RastaSaathi is a mission-critical platform designed to eliminate the "Golden Hour" delay in road accident response.

---

## 🛠️ Technical Stack

### **Frontend (The HUD)**
- **Framework:** React.js + Vite (for high-speed performance)
- **Animations:** Framer Motion (Glassmorphic HUD transitions)
- **Icons:** Lucide React (Medical-grade iconography)
- **Maps:** Google Maps React (Real-time victim/hospital tracking)
- **State Management:** React Hooks & Context API

### **Backend (The Intelligence)**
- **Runtime:** Node.js (Express Framework)
- **Database:** MongoDB Atlas (Geospatial `$near` indexing)
- **Real-time:** Socket.IO (planned for ambulance tracking)
- **Authentication:** JSON Web Tokens (JWT) & BCrypt hashing

### **AI Core (The Brain)**
- **Primary Model:** Anthropic Claude 3.5 Sonnet
- **Secondary Model:** Google Gemini 1.5 Flash (with rotation logic)
- **Utility:** Medical Triage, First-Aid Guidance, Severity Prediction

### **Connectivity (The Rescue)**
- **Messaging:** Twilio API (Dual-Channel: WhatsApp + SMS Alerts)
- **Mapping:** Google Maps API (Distance Matrix, Geocoding)
- **Hosting:** Vercel (Frontend) & Render (Backend)

---

## 🚀 Key Features

1. **Unified SOS Trigger:** Captures location and alerts contacts in < 1 second.
2. **AI Medic Assistant:** Instant first-aid instructions based on injury severity.
3. **Trauma Center Mapping:** Geospatial routing to 370+ equipment-verified hospitals.
4. **Dual-Channel Alerts:** Simultaneous WhatsApp and SMS delivery for 100% reliability.
5. **Premium Mobile HUD:** Fully responsive glassmorphic UI for emergency responders.
